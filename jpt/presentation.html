Introduction

  • Cultural Heritage: Newari (Nepal Bhasa) is one of the oldest and most significant indigenous languages of Nepal.
  • Accessibility Gap: Challenges in structured digital learning resources, especially for the Bhaktapur dialect.
  • Digital Preservation: Need for a dedicated platform that supports systematic learning using modern web tech.
Preserving Nepal Bhasa through Digital Innovation
1.1 Introduction Bhasabridge Slide 2

Problem Statement

Critical

Lack of structured, accessible, and interactive digital learning platforms.

  • Existing resources are informal, scattered, or limited to static textbooks.
  • Absence of centralized systems makes progress tracking and consistent learning difficult.
  • The language faces decline among urban youth due to lack of engaging digital tools.

Literature Review

  • Flashcard-Based Learning: Proven technique for vocabulary acquisition and memory retention.
  • LMS Integration: Research shows progress tracking increases learner motivation and accountability.
  • Full-Stack Choice: Flask (Backend) + React (Frontend) ensures responsive interfaces and scalable logic.

Database Design (ERD)

  • User Entity: Handles auth and profile.
  • Lesson & Quiz: 1:N Relationship (Lessons contain Quizzes).
  • Question: Weak entity dependent on Quiz ID.
  • UserProgress: Relational bridge tracking scores and completion.

Backup: SQLite Schema

-- Core Schema for BhasaBridge CREATE TABLE Users ( id INTEGER PRIMARY KEY, username VARCHAR(50) UNIQUE, password_hash TEXT, role VARCHAR(10) -- 'admin'/'user' ); CREATE TABLE Progress ( id INTEGER PRIMARY KEY, user_id INT, lesson_id INT, score FLOAT, FOREIGN KEY(user_id) REFERENCES Users(id) );
  • ACID Compliance: Ensured by SQLite engine.
  • Scalability: Ready for migration to PostgreSQL.
  • Authentication: JWT-based tokens integrated with Flask-SQLAlchemy.
